# Планировщик задач (TODO)

Дипломный проект — веб-сервер для планировщика задач, аналог TODO-листа.
Сервер отдаёт готовый фронтенд из папки `web` и реализует REST API для работы
с задачами: добавление, список, получение одной задачи, редактирование,
отметку о выполнении и удаление. Данные хранятся в базе SQLite.



## Задания со звёздочкой

Задания повышенной сложности не выполнялись. Реализованы только базовые
правила повторения `d` и `y`. Поиск, правила `w`/`m`, аутентификация и Docker
не делались.

## Запуск

```
go run .
```

По умолчанию сервер слушает порт 7540, открывать в браузере:

```
http://localhost:7540/
```

При старте, если файла `scheduler.db` нет, он создаётся автоматически вместе
с таблицей.

### Переменные окружения

Оба параметра необязательные:

- `TODO_PORT` — порт сервера (по умолчанию 7540)
- `TODO_DBFILE` — путь к файлу базы данных (по умолчанию `scheduler.db`)

Пример:

```
TODO_PORT=8000 go run .
```

## Тесты

Перед запуском тестов сервер должен быть запущен (в отдельном терминале).

```
go test ./tests
```
